# Operation KidZania 2026

Modern cute mobile-first mission tracker built with React, Vite, and TailwindCSS.

## Cara menjalankan

```bash
npm install --registry=https://registry.npmjs.org/ --no-audit --no-fund
npm run dev
```

Buka link lokal yang muncul, biasanya `http://localhost:5173`.

## Revisi Reward XP

Reward sekarang memakai spendable XP balance.

- EsKrim — 50 XP
- Bunga — 350 XP
- Makan Cantik — 650 XP
- Request — 1500 XP
- Mobil Listrik — 2002 XP

Jika reward dicentang, XP akan berkurang sesuai harga. Jika reward di-uncheck, XP dikembalikan.

## Secret Reward

Secret reward diubah menjadi approval panel dari Agent Wiyona:

- ACC untuk next trip/main lagi bersama Geo
- Decline jika belum disetujui

Semua progress tersimpan otomatis di localStorage.
